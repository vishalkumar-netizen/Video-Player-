## 🚀 About The Project

Ultra Pro Local Video Player is a fully browser-based, privacy-focused, open-source video player that allows users to play local video files directly in their browser without uploading anything.

---

## ✨ Features

* 🎥 Netflix-style custom control bar
* ⚡ FFmpeg.wasm support (MKV / AVI auto convert to MP4)
* 📱 Mobile double-tap seek gestures
* 🚀 Progressive Web App (Installable)
* 🔒 100% Client-side (No server upload)
* ⏩ Playback speed control (0.5x – 2x)
* 📺 Fullscreen support
* 📸 Screenshot capture
* 🎚 Custom progress bar
* 📦 Offline caching (Service Worker enabled)

---

## 🛠 How To Use

1. Open the website.
2. Click or drag & drop your video file.
3. If file format is unsupported (MKV/AVI), automatic conversion will begin.
4. Use custom control bar for playback control.
5. On mobile: Double tap left/right side to seek 10 seconds.
6. Click Install to add to Home Screen (PWA supported browsers).
